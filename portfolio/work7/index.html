不卖橘子的江先生

推荐电脑配置 快人一步

随着电脑技术的发展，大型3D游戏的精细度一定是越来越高的，因此配置的要求也会越来越高，尤其是这种特别吃细节的FPS游戏，一个像素可能就决定了能否看到敌人，因此配置自然是越高越好。

官方公布的配置要求，其实仅仅只能勉强维持游戏的运行，而中途会碰到各种卡顿，画面显示不完全或者甚至无画面只有声音的尴尬情况。

因此我们从上到下各个介绍一下到底要什么样的配置，才能流畅地运行绝地求生这款游戏：

  1. 操作系统：64-bit Windows 7, Windows 8.1, Windows 10；处理器：Intel Core i3-4340 / AMD FX-6300
  2. 内存:8 GB RAM；图形（显卡）:nVidia GeForce GTX 660 2GB / AMD Radeon HD 7850 2GB；存储空间:需要 30 GB 可用空间

绝地求生这款游戏可以达到144的平均帧，这是非常高的帧数，但是并不是必须的，人眼并没有那么强的识别能力，基本能达到70以上的平均帧，游戏画面已经很精细且流畅了。因此在不考虑AMD显卡的情况下，GTX 1060 6G的显卡已经完全够了，平均帧达到90多，最低帧也有60多。游戏中最低帧的时候一般在画面较静止的时候，60多也已经不错了。 有人问，为什么硬盘也会对游戏的运行造成影响？不就是一个存储设备么？是的，就是一个存储设备，可是如果这个存储设备的读写能力不强，将会影响游戏数据的输入和输出，从而影响游戏的运行。好比硬盘是仓库，如果这个仓库大门太小，员工效率太低，哪怕仓库内部空间像足球场一样大，出货速度还是很慢。因此在选择硬盘的时候，最好能选择一个固态硬盘。相比机械硬盘，固态硬盘的读写速度会有大幅度的提升。

英雄联盟需要配置什么样的主机

英雄联盟是现在比较火的电竞游戏，团队合作，相互配合，找几个朋友就能开黑 ，还是许多人联络感情的一种方式。但是家里电脑玩游戏很卡，会是什么感受？相信很多人都有体会。那简直生不如死，正在打团，突然就黑屏，蓝屏，整个人心理都要崩溃，恨不得把电脑给砸了 。

官方所说的最低配置，大家基本可以不用考虑。为什么？因为最低配置是可以玩，但是在不考虑流畅度的情况下，你可以玩，后果是，很卡，很不流畅，生不如死，不如不玩。

  • CPU I3 2100是英特尔的早期产品，在2014年已经停产，某宝二手市价220元左右。属于高频双核四线程。 （推荐配置六代I5CPU）
  • 内存条4G1333 DDR3 根据品牌不一样，售价在120元左右。建议使用颗粒度饱满的三星或宇瞻双面宽条内存条，劣质内存条对电脑主机的速度影响是非常大的。（推荐使用DDR4内存）

显卡是游戏的重中之重，要想游戏清楚，化画流畅必须要配张好显卡。官方推荐1G显存是可以。 为什么这么说？ 因为I3 2100 这款CPU是带显存的，再加上显卡的显存，是够用的。但是我们还是推荐使用2G显存的显卡。
